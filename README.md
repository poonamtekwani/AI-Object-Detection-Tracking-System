# 🚀 AI Object Detection & Tracking System using Azure

🔍 Transforming visual data into intelligent insights using cloud-based AI

---

## 🧠 Project Overview

This project implements an end-to-end **object detection and tracking system** using Microsoft Azure services. It detects objects in images and video frames, visualizes predictions, and analyzes object movement over time.

---

## ⚙️ Tech Stack

* Python
* Microsoft Azure Computer Vision
* Azure Custom Vision
* OpenCV
* Matplotlib
* Jupyter Notebook

---

## 🔍 Key Features

* Object detection using Azure Computer Vision API
* Custom object detection model using Azure Custom Vision
* Video frame extraction using OpenCV
* Bounding box visualization with confidence scores
* Object movement tracking across frames
* Exported tracking data to CSV for analysis

---

## 📊 Results

* Achieved high detection accuracy (>85%) for common objects
* Fast prediction time (~1–2 seconds per frame)
* Successfully processed both images and video data

---

## ⚠️ Limitations

* Basic tracking (no unique object IDs)
* Performance may vary in different environments

---

## 💡 Conclusion

This project demonstrates how cloud AI services can be used to build practical computer vision systems for real-world applications.
